What is the Document Object Model (DOM)?

The Document Object Model (DOM) is programming interface for web documents. It represents the structure of an HTML document as a tree of objects, allowing scripts to manipulate content dynamically. DOM is essential for dynamic web development. It acts as the bridge between JS and the webpage, allowing developers to manipulate content in real time.

How do you select an element from the DOM?

There are five different DOM selection methods to access and manipulate HTML element in Javascript. Those are used for different purpose to select HTML element.
1. Select by ID - document.getElementById("id")
2. Select by Class - document.getElementByClass("class")
3. Select by Tag - document.getElementByTagName("tag")
4. Select single element by Query Selector - document.querySelector("selector")
5. Select multiple elements - document.querySelectorAll("selector")

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JS where a paren element listens for events on its child elements using event bubbling. Instead of adding event listeners to multiple child element, you add a single event listener to the parent. Here is an example of event delegation. Here instead of adding listeners to each <li>, we add a listener to the <ul> and use the event.target to detect which <li> was clicked. 

              <ul id="list">
                  <li>Item 1</li>
                  <li>Item 2</li>
                  <li>Item 3</li>
                </ul>

                <script>
                  document.getElementById("list").addEventListener("click", function (event) {
                    if (event.target.tagName === "LI") {
                      alert(event.target.innerText + " clicked");
                    }
                  });
                </script>

Why is event delegation useful?
1. Reduce memory usage
2. Reduce the number of event listeners
3. Simplify the code

When to use event delegation?
1. When a large number of child elements
2. When handling events on a group of elements
